# Course Management Service

## Overview
CourseManagementService is a Java-based application designed to manage courses efficiently. It leverages Spring Boot for easy development and Redis for caching to enhance performance.

## Features
- Manage course entities (create, read, update, delete)
- Implement Redis caching for improved performance
- Docker support for easy deployment

## Technology Stack
- **Java**: Programming language
- **Spring Boot**: Framework for building the application
- **Redis**: In-memory data structure store for caching
- **Maven**: Build automation tool

### Prerequisites
- Java 11 or higher
- Maven
- Docker (for containerization)
